2. Objeto y alcance

El objeto de la Aplicación es proporcionar a las Usuarias un sistema de alerta discreta diseñado específicamente para víctimas de violencia de género. La Aplicación permite activar alertas de emergencia de forma discreta desde un smartwatch mediante un gesto predefinido, sin necesidad de interactuar visiblemente con el dispositivo, priorizando la rapidez, discreción y seguridad de la Usuaria.

La Aplicación se compone de dos elementos principales:

a) Aplicación para smartwatch (Wear OS): Constituye el elemento central de activación del sistema de emergencia. Permite a la Usuaria iniciar una alerta mediante un gesto predefinido con la muñeca, sin necesidad de tocar la pantalla. El diseño de la interfaz está orientado a disimular la actividad de la Aplicación. Una vez activada la alerta, el smartwatch puede iniciar la grabación de audio del entorno. La Usuaria puede cancelar la alerta si la activa por error o si la situación de riesgo desaparece.

b) Aplicación móvil (Android): Actúa como centro de configuración del sistema. Permite la creación de una cuenta personal, la concesión de permisos necesarios, un tutorial de gestos de activación, la vinculación con el smartwatch, la definición de contactos de confianza que recibirán las alertas (el/los “Tercero/s Designado/s”) y la ubicación de la Usuaria, la configuración de gestos y palabras clave de desactivación, mecanismos de discreción (modo camuflaje con interfaz alternativa protegida por PIN, gestión de notificaciones discretas, ocultación del uso real de la Aplicación) y el acceso al historial de alertas y grabaciones de audio.

La Aplicación incorpora un sistema de camuflaje diseñado para proteger a la Usuaria en caso de que el presunto agresor o un tercero no autorizado acceda al dispositivo móvil. La Aplicación muestra una interfaz alternativa e inocua (como una aplicación de salud, entrenamiento o similar) cuando no se ha introducido el PIN de acceso. Una vez introducido el PIN, se revela la aplicación real con todas sus funcionalidades, historial y evidencias. En el smartwatch, la pantalla permanece en negro durante la activación para no despertar sospechas.
